Let $\mathcal{P}$ be the set of vectors defined by \[ \mathcal{P} = \left\{\begin{pmatrix} a \\ b \end{pmatrix} \, \middle\vert \, 0 \le a \le 2, 0 \le b \le 100, \, \text{and} \, a, b \in \mathbb{Z}\right\}. \] Find all $\vec{v} \in \mathcal{P}$ such that the set $\mathcal{P}\setminus\{\vec{v}\}$ obtained by omitting vector $\vec{v}$ from $\mathcal{P}$ can be partitioned into two sets of equal size and equal sum.

2018 Putnam Exam, Problem B1
